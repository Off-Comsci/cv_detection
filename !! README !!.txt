### ขั้นตอนการใช้งาน ###
1. รันไฟล์  A.py 
    - เพื่อทำการจดจำใบหน้า โดยกรอกรหัสนักเรียน (กรอกเป็นตัวเลข 5 หลักเท่านั้น)
    - จากนั้นจะมีหน้าต่างกล้องขึ้นมาเพื่อจดจำใบหน้า ให้ทำหน้าหลาย ๆ ลักษณะ จนกว่าหน้าต่างกล้องจะหายไป
2. รันไฟล์ B.py
    - เพื่อ training รอจนกว่าขึ้นสถานะ success
3. รันไฟล์ C.py
    - เพื่อวิเคราะห์ใบหน้า
** หมายเหตุ 
    - เมื่อตรวจพบใบหน้าตรงกับที่บันทึกไว้ระบบจะแสดงผลตามรหัสนักเรียนที่ทำการบันทึกไว้ในขั้นตอนที่ 1
    - หากต้องการให้แสดงชื่อนักเรียนแทนรหัสให้ไปแก้ที่ไฟล์ idtoname.py โดยเพิ่มรหัสนักเรียนกับชื่อลงไป
    - ระบบรองรับการแสดงผลเป็นภาษาอังกฤษเท่านั้น

### Module requirements ###
cv2	=> pip install opencv-python
numpy 	=> pip install numpy
PLT	=> pip install Pillow
os	=> pip install os-win // ปกติจะติดตั้งมากับpythonอยู่แล้ว

*** หากรันไฟล์แล้วกล้องไม่ขึ้นให้ติดตั้ง pip install opencv-contrib-python
